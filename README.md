# MedHelp - Healthcare & Medicine Delivery App

MedHelp is a comprehensive Android application designed to provide users with a seamless experience for ordering medicines, booking lab tests, and managing their healthcare needs from the comfort of their home.

## 📺 Project Demo
You can watch the full demo of the application here:
[**Watch Demo Video**](https://drive.google.com/file/d/1BJeUlk4sXe4C7ipgPdRFutX9IzMJKy_a/view?usp=sharing)

## ✨ Features

- **User Authentication:** Secure Login and Registration system with Firebase, including profile picture uploads and password recovery.
- **Medicine Marketplace:** Browse a wide range of medicines and healthcare products with detailed descriptions.
- **Cart & Checkout:** Add products to cart, manage quantities, and proceed through a streamlined checkout process.
- **Lab Tests:** Explore and book various diagnostic lab tests.
- **Offers & Discounts:** Dedicated section for the latest deals and coupons.
- **Order Tracking:** Keep track of your health orders and history.
- **Profile Management:** Update personal details, email, and security settings.

## 🛠️ Tech Stack

- **Language:** Java / Kotlin
- **Database/Backend:** Firebase Authentication, Realtime Database/Firestore
- **UI Design:** Material Design Components, XML
- **Network:** Retrofit (for API calls)
- **Image Loading:** Glide / Picasso

## 🚀 Getting Started

### Prerequisites
- Android Studio
- Android SDK 24+
- A Firebase project linked to the app (`google-services.json` included)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/rohanrathod5999/MEDHELP.git
   ```
2. Open the project in **Android Studio**.
3. Let Gradle sync and download dependencies.
4. Run the app on an Emulator or Physical Device.
